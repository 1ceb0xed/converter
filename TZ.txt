### Техническое задание (ТЗ)

Название проекта: SPA для конвертирования валют

Цель: Разработать одностраничное приложение (SPA) для удобной конвертации валют с использованием Vue3.

Основные требования:

1. Структура приложения:
   - Хедер: +
     - Должны быть ссылки на следующие страницы:
       - Главная ("/")
       - Конвертация ("/convert")
     - Dropdown для выбора основной валюты сайта (USD, EUR, RUB).
   
   - Страницы: 
     - Главная страница ("/"):
       - Отображение курсов валют относительно выбранной основной валюты.
        Например, если основная валюта — RUB, показывать:
         - 1 USD = 91.45 RUB
         - 1 EUR = 101.74 RUB

     - Страница конвертации ("/convert"): +
       - Форма с двумя строками:
         - Каждая строка должна содержать:
           - Dropdown для выбора валюты (USD, EUR, RUB)
           - Числовой Input
         - При изменении значения в любом Input автоматически пересчитывать
          и обновлять значение в другом Input.
         - Значения округляются до 2 знаков после запятой.








2. Подключение к API:
   - Использовать API для получения курсов валют: 
   GET https://status.neuralgeneration.com/api/currency или любое другое открытое API.
   
3. Технические параметры:
   - Использовать Vue3.
   - Разделение на компоненты.
   - Обеспечить хорошее взаимодействие между компонентами.
   - Поддержка валют: USD, EUR, RUB.

4. Дополнительные требования:
   - Обработка ошибок и проверка входных данных в Input (например, предотвращение ввода текстовых значений).
   - Возможность легко изменять основной валютный API, если потребуется.
   - Дизайн остаётся на усмотрение разработчика, приветствуется интуитивно понятный и приятный интерфейс.

5. Документация и развертывание:
   - Подготовить инструкцию по запуску и развертыванию приложения.
   - Документация кода и возможность дальнейшего расширения функциональности.

Примечания:
- Рекомендуется обратите внимание на пользовательский опыт при разработке интерфейса.
- Возможно использование сторонних библиотек для Dropdown или Input на усмотрение разработчика.


Nuxt 3, TypeScript, Vuetify
